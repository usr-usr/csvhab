# CSV家計簿（開発中）
本プロジェクトは、Androidアプリ開発における「変更に強く、メンテナンス性の高い設計」の実践を主目的としています。 日常的に利用しているPayPay等の決済アプリから出力されるCSVを活用し、「最小限の手間で家計を管理する」ことを目指した個人開発プロジェクトです。

## 概要/学習内容
* Android Studioの習得
* Kotlinの習得
* XMLの習得
* クリーンな設計の学習
  * 関心の分離
  * ドメイン駆動設計（DDD）の試行
  * 保守性の向上

## 現在の主な機能
* 手入力による家計簿登録
* 費目振分けルールの追加/編集
