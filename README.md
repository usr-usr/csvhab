# CSV家計簿（開発中）
本プロジェクトは、Androidアプリ開発における**「変更に強く、メンテナンス性の高い設計」の実践**を主目的としています。 日常的に利用しているPayPay等の決済アプリから出力されるCSVを活用し、「最小限の手間で家計を管理する」ことを目指した個人開発プロジェクトです。

## 概要/学習内容
* Android Studioの習得
* Kotlinの習得
* 変更/拡張に強い設計
* DRY原則
* 依存先の限定
* Ui/ビジネスロジック/データベースの分離
* ドメインモデルについて

## 進捗
手入力と取引相手から費目振り分けるペアの追加/単一編集/複数編集の実装．
次に費目の追加/単一編集/複数編集の実装．
